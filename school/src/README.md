# How I Code with AI

This series walks through how I currently work with AI coding agents — broken into three lessons of increasing complexity and power.

Each lesson uses the same boilerplate todo app as its starting point. You'll see the same feature added at each level so the comparison is apples to apples: what changes is the methodology, the structure, and how much you trust the AI to self-review its own work.

Pick a level and start there. You don't need to do them in order, but the concepts build on each other.
